# Simple-chat

### Обзор
* Описание
* Технологии

**Описание**

Чат-приложение с клиентской и серверной частью.

Функионал:
* Логин пользователя
* Несколько чат-комнат
* Пользователи видят участников комнаты в данный момент
* Сообщения промаркированы именем автора, временем отправки

**Технологии**

* Node.js/Express
* JS/React/CSS
* Socket.io
